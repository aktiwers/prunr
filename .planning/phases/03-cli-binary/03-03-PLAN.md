<task type="checkpoint:human-verify" gate="blocking">
  <name>Task 1: Human verification of CLI behavior with real images</name>
  <what-built>
Full prunr CLI: `prunr remove` subcommand with single-image and batch modes, model selection,
parallelism control, progress display (stage spinner + MultiProgress), overwrite protection, and
exit codes 0/1/2.
  </what-built>
  <how-to-verify>
First, build the binary in release mode with dev-models (or ensure models are fetched):

```bash
cd /path/to/prunr
# If models not fetched yet:
cargo xtask fetch-models

# Build with embedded models
cargo build --release -p prunr-app

# Or build with dev-models for faster iteration
cargo build -p prunr-app --features prunr-models/dev-models
```

Then run each check with a real JPEG or PNG (any photo with a clear subject):

**Check 1 — Single image, basic output:**
```bash
./target/debug/prunr remove photo.jpg -o out_single.png
# Expected: stage spinner shows (Decoding... → Inferring... → Applying alpha...)
# Expected: "done: photo.jpg -> out_single.png" printed
# Expected: exit code 0
echo "Exit: $?"
# Open out_single.png — must have transparent background, subject preserved
```

**Check 2 — Output naming convention:**
```bash
./target/debug/prunr remove photo.jpg
# Expected: photo_nobg.png created alongside photo.jpg
# Expected: no -o flag needed
```

**Check 3 — Overwrite protection:**
```bash
./target/debug/prunr remove photo.jpg -o out_single.png
# Expected: "error: output 'out_single.png' already exists. Use --force to overwrite."
# Expected: exit code 1
./target/debug/prunr remove photo.jpg -o out_single.png --force
# Expected: succeeds and overwrites
```

**Check 4 — Model selection:**
```bash
./target/debug/prunr remove photo.jpg -o out_silueta.png --model silueta
./target/debug/prunr remove photo.jpg -o out_u2net.png --model u2net
# Expected: both succeed. Output files may differ visually (u2net = higher quality edges).
```

**Check 5 — Batch mode with --output-dir:**
```bash
mkdir -p /tmp/prunr-test-out
./target/debug/prunr remove photo.jpg photo2.jpg --output-dir /tmp/prunr-test-out/
# Expected: MultiProgress renders — overall bar + per-image spinners
# Expected: completion lines: "✓ photo.jpg (1.2s)"  "✓ photo2.jpg (1.1s)"
# Expected: "2 succeeded, 0 failed."
# Expected: exit code 0
echo "Exit: $?"
ls /tmp/prunr-test-out/
# Expected: photo_nobg.png  photo2_nobg.png
```

**Check 6 — --jobs parallelism:**
```bash
./target/debug/prunr remove photo.jpg photo2.jpg photo3.jpg \
  --output-dir /tmp/prunr-test-out2/ --jobs 2
# Expected: no crash, all 3 outputs written
```

**Check 7 — --quiet mode:**
```bash
./target/debug/prunr remove photo.jpg -o out_quiet.png --quiet
# Expected: zero stdout output
# Expected: exit code 0
```

**Check 8 — Exit codes:**
```bash
./target/debug/prunr remove nonexistent.jpg --output-dir /tmp/
echo "Exit code for missing file: $?"  # Expected: 1

./target/debug/prunr remove photo.jpg nonexistent.jpg --output-dir /tmp/prunr-ec/
echo "Exit code for partial failure: $?"  # Expected: 2
```

**What to look for in output images:**
- Subject is preserved with correct edges
- Background is fully transparent (checkerboard in image viewer, not white/black)
- PNG file is valid (image viewer opens it without error)
  </how-to-verify>
