2. Implement `BgPrunrApp::new(cc: &eframe::CreationContext)`:
   - Spawn worker via `worker::spawn_worker(cc.egui_ctx.clone())`
   - Initialize clipboard via `arboard::Clipboard::new().ok()` (Option -- don't crash if clipboard unavailable)

3. Implement `eframe::App` trait with **`logic()` + `ui()` split** (NOT deprecated `update()`):

**`fn logic(&mut self, ctx: &egui::Context, _frame: &mut eframe::Frame)`:**
   - Poll worker channel: `while let Ok(msg) = self.worker_rx.try_recv()` handling:
     - `WorkerResult::Progress(stage, pct)` -> update progress_stage (map ProgressStage to display string per UI-SPEC: Decode/Resize/Normalize -> "Preprocessing", Infer -> "Inferring", Postprocess/Alpha -> "Applying alpha"), update progress_pct
     - `WorkerResult::Done(result)` -> decode rgba_bytes to RgbaImage via `image::load_from_memory(&result.rgba_bytes)`, convert to `egui::ColorImage::from_rgba_unmultiplied()`, call `ctx.load_texture("result", color_image, TextureOptions::default())` to store as `result_texture`, set `active_backend` from `result.active_provider`, set state to `AppState::Done`, set `status_text` to `"Done"`
     - `WorkerResult::Cancelled` -> set state to `AppState::Loaded`, set `status_text` to `"Cancelled"`
     - `WorkerResult::Error(msg)` -> set state to `AppState::Loaded`, set `status_text` to error message per UI-SPEC: "Processing failed. Try a different image or restart the app."
   - Handle drag-and-drop: `let dropped = ctx.input(|i| i.raw.dropped_files.clone());` -- for each file with a path, call `self.handle_open_path(path)`
   - Handle keyboard shortcuts via `ctx.input(|i| { ... })`:
     - `i.modifiers.command && i.key_pressed(Key::O)` -> open file dialog (rfd)
     - `i.modifiers.command && i.key_pressed(Key::R)` -> dispatch inference (if state is Loaded or Done)
     - `i.modifiers.command && i.key_pressed(Key::S)` -> save dialog (if state is Done)
     - `i.modifiers.command && i.key_pressed(Key::C)` -> copy to clipboard (if state is Done)
     - `i.key_pressed(Key::Escape)` -> if Processing: send cancel; if show_shortcuts: close overlay
     - `i.key_pressed(Key::QuestionMark)` -> toggle show_shortcuts
   - Update window title: if `loaded_filename` changed, send `ctx.send_viewport_cmd(ViewportCommand::Title(...))`
     - Empty: "BgPrunR"
     - With file: "BgPrunR \u{2014} {filename}" (em dash)
   - Clear temporary status text after ~2 seconds (track a frame counter or instant)

**`fn ui(&mut self, ui: &mut egui::Ui, _frame: &mut eframe::Frame)`:**
   - `egui::TopBottomPanel::top("toolbar").exact_height(theme::TOOLBAR_HEIGHT).show_inside(ui, |ui| toolbar::render(ui, self));`
   - `egui::TopBottomPanel::bottom("statusbar").exact_height(theme::STATUS_BAR_HEIGHT).show_inside(ui, |ui| statusbar::render(ui, self));`
   - `egui::CentralPanel::default().show_inside(ui, |ui| canvas::render(ui, self));`
   - If `self.show_shortcuts`: `shortcuts::render(ui.ctx());`
